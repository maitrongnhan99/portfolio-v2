# GEMINI.md

This file provides guidance to Gemini when working with code in this repository.

**The single source of truth for this project is [`CLAUDE.md`](./CLAUDE.md).**

Read `CLAUDE.md` for everything: project overview, development commands, high-level
architecture, route groups, the project data flow, the AI assistant / RAG stack, API
surfaces, styling and design system, testing shape, versioning, and environment variables.

Also follow the UI design rule in `CLAUDE.md`: read `DESIGN.md` before making meaningful
UI changes and treat it as the source of truth for the visual language.

Do not duplicate guidance here - keeping a second copy in sync is the problem this file
avoids. Update `CLAUDE.md` instead; `AGENTS.md` (Codex) is the near-verbatim mirror to keep
aligned when architecture or commands change.
